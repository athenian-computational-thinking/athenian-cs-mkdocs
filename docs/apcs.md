# APCS Details

## gradlew Commands

Create `gradlew` with:
```
gradle wrapper
```

Build project with:
```
./gradlew assemble
```

Clean project with:
```
./gradlew clean
```

Build and test project with:
```
./gradlew build
```

Build uberjar file with:
```
./gradlew mymain
```

Run uberjar file with:
```
java -jar build/libs/mymain.jar
```
